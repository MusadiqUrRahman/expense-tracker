● Based on what I've read, Spendly is a personal expense tracker web app. Here's
   what it does:

  Core Purpose:
  - Track daily expenses and understand spending patterns

  What's Built So Far:
  - Landing page — marketing page with hero section, feature highlights, and CTA
  - Register page — form to create account (name, email, password)
  - Login page — form to sign in (email, password)
  - Base layout — navbar with brand "Spendly" + footer

  What's Planned (placeholder routes in app.py):
  - Database — SQLite setup with get_db(), init_db(), seed_db()
  - Authentication — login/register POST handlers, logout, sessions
  - Profile page — user profile view
  - Expense CRUD — add, edit, and delete expenses
  - Category breakdowns — spending by category (Bills, Food, Health, Transport)
  - Date filtering — view expenses by time periods

  Tech Stack:
  - Backend: Flask (Python 3.12)
  - Database: SQLite
  - Frontend: Jinja2 templates, custom CSS (DM Serif Display + DM Sans fonts)
  - Testing: Pytest + Pytest-Flask

  It's a course project built step-by-step — the UI is ready but most backend
  logic (database, auth, expense management) still needs to be implemented.
