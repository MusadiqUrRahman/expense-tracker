# ⚡ /exit — Claude Code Command

## 📌 Purpose
`/exit` is used to exit the REPL (interactive session) in Claude Code.

---

## ⚙️ What It Does
- Closes the current interactive session  
- Stops the REPL (Read–Eval–Print Loop)  
- Ends communication with Claude  
- Returns control back to your terminal  

---

## 🚀 Usage
/exit

---

## 💡 When to Use
- When you finish your coding session  
- When you want to stop the REPL  
- When you want to leave interactive mode  

---

## 🧠 Pro Tip
You can also press **Ctrl + D** or use `/quit` to exit the session. :contentReference[oaicite:0]{index=0}  

---

## ⚠️ Notes
- This command completely ends the session  
- You need to start Claude Code again to continue  

---

## 📚 Summary
`/exit` = Exit REPL and close session
