# ⚡ /compact — Claude Code Command

## 📌 Purpose
`/compact` is used to **reduce conversation size while keeping a useful summary in context**.

---

## ⚙️ What It Does
- Clears old conversation messages  
- Keeps an important summary of past work  
- Frees up context (tokens)  
- Improves performance and accuracy  

---

## 🚀 Usage
/compact

### With Instruction (Optional)
/compact focus on authentication system

---

## 💡 How It Works
- Removes detailed old messages  
- Creates a short summary of what happened  
- Keeps important context only  
- Gives Claude more space to continue working  

---

## 💡 When to Use
- When session becomes very long  
- When Claude becomes slow or confused  
- When you see high token usage  
- Before starting a new phase of work  

---

## 🧠 Pro Tip
Use `/compact` regularly in long sessions to keep Claude fast and accurate.

---

## ⚠️ Notes
- It does NOT delete everything completely  
- It keeps a summarized version of your work  
- Helps prevent context overflow  

---

## 📚 Summary
`/compact` = Clean chat but keep summary
