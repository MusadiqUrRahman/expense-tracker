# ⚡ /ultraplan — Claude Code Command

## ⚠️ Important Note
- `/ultraplan` is an advanced planning feature (cloud-based)  
- It sends your task to Claude Code on the web (browser)  
- You can review, comment, and edit the plan before execution  
- Best for complex and large projects  

---

## 📌 Purpose
`/ultraplan` is used to create a **detailed and structured plan in the cloud** and review it before executing.

---

## ⚙️ What It Does
- Moves planning from terminal to cloud (browser)  
- Analyzes your full codebase  
- Creates a detailed step-by-step plan  
- Lets you review and comment on specific sections  
- Allows execution in cloud or terminal  

---

## 🚀 Usage
/ultraplan your-task

Example:
/ultraplan build authentication system

---

## 🧠 How It Works
1. You run `/ultraplan`  
2. Claude sends task to cloud environment  
3. Your project is analyzed remotely  
4. A structured plan is created  
5. You open it in browser  
6. Review, comment, and request changes  
7. Approve → execute in cloud or terminal  

👉 Planning happens in browser, not terminal :contentReference[oaicite:0]{index=0}  

---

## 🔍 Key Features

### 🌐 Cloud Planning
- Plan is created in web environment  
- Terminal remains free  

---

### 💬 Section Comments
- You can comment on specific parts  
- Easy to refine plan  

---

### 🔄 Revision System
- Ask Claude to update plan  
- Iterate until perfect  

---

### ⚙️ Flexible Execution
- Run in cloud  
- Or send back to terminal  

---

## 💡 When to Use
- Large or complex projects  
- Multi-file changes  
- System redesign  
- Advanced debugging  
- Team-level workflows  

---

## 🧠 Pro Tip
Use `/ultraplan` when `/plan` feels too simple or hard to review.

---

## ⚠️ Notes
- Requires Claude Code web integration  
- May need GitHub repository  
- Takes more time than `/plan`  
- Best for serious development work
