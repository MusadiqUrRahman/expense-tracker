# ⚡ /stats — Claude Code Command

## 📌 Purpose
`/stats` is used to **view your usage statistics and activity** in Claude Code.

---

## ⚙️ What It Does
- Shows detailed usage statistics  
- Displays your activity history  
- Tracks how you use Claude over time  
- Helps understand your working patterns  

---

## 🚀 Usage
/stats

---

## 💡 What You Will See
- Total number of requests  
- Token usage (input/output)  
- Activity over time  
- Model usage breakdown (Opus, Sonnet, Haiku)  
- Session activity details  

---

## 💡 When to Use
- When you want detailed analytics of your usage  
- When tracking your work habits  
- When optimizing performance and cost  
- When analyzing model usage  

---

## 🧠 Pro Tip
Use `/stats` together with `/usage` to fully understand both **limits and detailed activity**.

---

## ⚠️ Notes
- `/usage` = limits and remaining quota  
- `/stats` = detailed analytics and history  
- Helps improve efficiency and planning  

---

## 📚 Summary
`/stats` = Detailed usage analytics  

---
