# ⚡ claude -r — Claude Code Command

## 📌 Purpose
`claude -r` is used to **resume a previous session** in Claude Code.

---

## ⚙️ What It Does
- Opens a past session using its ID  
- Restores full conversation history  
- Continues work from where you left off  
- Keeps previous context and progress  

---

## 🚀 Usage
claude -r "session-id" "your message"

Example:
claude -r "abc123" "continue this task"

---

## 💡 When to Use
- When you want to continue old work  
- When you closed terminal by mistake  
- When working on long projects  
- When you don’t want to lose context  

---

## 🧠 Pro Tip
Combine with `-p` to run and exit quickly:
claude -r "abc123" -p "run tests"

---

## ⚠️ Notes
- You need the correct session ID  
- If session not found, it will not resume  
- Sessions are stored automatically  

---

## 📚 Summary
`claude -r` = Resume previous session
