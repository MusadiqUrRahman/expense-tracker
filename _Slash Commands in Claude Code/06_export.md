# ⚡ /export — Claude Code Command

## 📌 Purpose
`/export` is used to **save the current conversation to a file or clipboard**.

---

## ⚙️ What It Does
- Exports full conversation history  
- Saves your session as a file  
- Can copy content to clipboard  
- Helps you keep a record of your work  

---

## 🚀 Usage
/export

---

## 💡 When to Use
- When you want to save your work  
- When sharing conversation with others  
- When creating documentation or notes  
- When backing up your session  

---

## 🧠 Pro Tip
Use `/export` after finishing a task to keep a clean record of your work.

---

## ⚠️ Notes
- Exports the current session only  
- Format may vary (file or clipboard)  
- Useful for long or important conversations  

---

## 📚 Summary
`/export` = Save conversation
