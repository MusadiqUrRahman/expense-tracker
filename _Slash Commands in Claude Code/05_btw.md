# ⚡ /btw — Claude Code Command

## 📌 Purpose
`/btw` is used to **ask a quick side question without stopping your current task**.

---

## ⚙️ What It Does
- Lets you ask a question while Claude is working  
- Does NOT stop your main task  
- Does NOT save the question in chat history  
- Shows answer in a temporary popup/overlay  
- Keeps your main conversation clean  

---

## 🚀 Usage
/btw your question here

Example:
/btw what does this function return?

---

## 💡 When to Use
- When you have a quick question during work  
- When Claude is running a long task  
- When you don’t want to interrupt workflow  
- When you want to avoid extra chat history  

---

## 🧠 Pro Tip
Use `/btw` for **small questions only**.  
For big tasks or changes, use normal prompts instead.  

---

## ⚠️ Notes
- Works like a temporary side chat  
- Answer is not saved permanently  
- Cannot run tools or edit files  
- Only uses current session context  

---

## 📚 Summary
`/btw` = Ask quick question without interrupting
