# ⚡ /config — Claude Code Command

## 📌 Purpose
`/config` is used to open the configuration panel in Claude Code.

---

## ⚙️ What It Does
- Opens settings panel  
- Lets you change Claude Code preferences  
- Allows model and behavior configuration  
- Controls how Claude works in your environment  

---

## 🚀 Usage
/config

---

## 💡 What You Can Configure
- Default model (Opus, Sonnet, Haiku)  
- Output style and behavior  
- Tool permissions  
- Environment settings  
- Session preferences  

---

## 💡 When to Use
- When you want to change settings  
- When switching models permanently  
- When customizing Claude behavior  
- When setting up your workflow  

---

## 🧠 Pro Tip
Use `/config` at the start of your project to set everything according to your needs.

---

## ⚠️ Notes
- Changes may apply to future sessions  
- Helps personalize your experience  
- Important for advanced users  

---

## 📚 Summary
`/config` = Open settings panel
