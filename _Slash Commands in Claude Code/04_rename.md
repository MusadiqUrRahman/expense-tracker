# ⚡ /rename — Claude Code Command

## 📌 Purpose
`/rename` is used to **change the name of the current conversation (session)**.

---

## ⚙️ What It Does
- Renames your current session  
- Helps you organize conversations  
- Makes it easier to find sessions later  
- Updates the session title  

---

## 🚀 Usage
/rename new-name

Example:
/rename auth-system-fix

---

## 💡 When to Use
- When you want better organization  
- When working on multiple tasks  
- When saving sessions for later use  
- When the default name is unclear  

---

## 🧠 Pro Tip
Use clear and meaningful names like:
- bug-fix-login  
- api-integration  
- ui-update  

👉 This helps you quickly find your work later  

---

## ⚠️ Notes
- Only renames the current session  
- Does not change your code or files  
- Useful for long-term projects  

---

## 📚 Summary
`/rename` = Change session name
