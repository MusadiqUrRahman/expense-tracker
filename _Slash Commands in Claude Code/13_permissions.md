# ⚡ /permissions — Claude Code Command

## 📌 Purpose
`/permissions` is used to **manage allow and deny rules for tools** in Claude Code.

---

## ⚙️ What It Does
- Controls which tools Claude can use  
- Allows or blocks specific actions  
- Manages access to files, commands, or system tools  
- Improves security and control  

---

## 🚀 Usage
/permissions

---

## 💡 What You Can Control
- Allow or deny file access  
- Allow or block terminal commands  
- Control tool usage (read/write/edit)  
- Set rules for safe execution  

---

## 💡 When to Use
- When you want more control over Claude actions  
- When working on sensitive projects  
- When restricting unsafe commands  
- When setting security rules  

---

## 🧠 Pro Tip
Allow only the tools you need and block everything else for better safety.

---

## ⚠️ Notes
- Helps protect your system and data  
- Useful for advanced users  
- Misconfiguration may block useful actions  

---

## 📚 Summary
`/permissions` = Control tool access

## 🇵🇰 اردو خلاصہ
یہ کمانڈ Claude Code میں tools اور actions کی اجازت (allow) اور پابندی (deny) کو کنٹرول کرنے کے لیے استعمال ہوتی ہے۔  
اس کے ذریعے آپ یہ طے کرتے ہیں کہ Claude کون سی چیزیں استعمال کر سکتا ہے اور کون سی نہیں، جیسے فائلز تک رسائی، کمانڈز چلانا یا سسٹم کے ساتھ interaction۔  
یہ خاص طور پر اس لیے ضروری ہے تاکہ آپ اپنے سسٹم، ڈیٹا اور پروجیکٹ کو محفوظ رکھ سکیں اور Claude صرف وہی کام کرے جو آپ اجازت دیں۔  
سادہ الفاظ میں، یہ کمانڈ آپ کو مکمل کنٹرول دیتی ہے کہ Claude کیا کر سکتا ہے اور کیا نہیں، تاکہ آپ محفوظ اور منظم طریقے سے کام کر سکیں۔
