# ⚡ /insights — Claude Code Command

## 📌 Purpose
`/insights` is used to **generate a detailed report about your Claude Code sessions**.

---

## ⚙️ What It Does
- Analyzes your past sessions  
- Shows patterns in your usage  
- Highlights your working behavior  
- Provides insights to improve efficiency  
- Summarizes how you use Claude Code  

---

## 🚀 Usage
/insights

---

## 💡 What You Will Get
- Session summaries  
- Usage patterns (how often you use Claude)  
- Model usage breakdown (Opus, Sonnet, Haiku)  
- Performance insights  
- Suggestions for better workflow  

---

## 💡 When to Use
- When you want to understand your usage deeply  
- When improving your workflow  
- When analyzing productivity  
- When optimizing cost and performance  

---

## 🧠 Pro Tip
Use `/insights` regularly to learn how you work and improve your efficiency over time.

---

## ⚠️ Notes
- Gives analysis, not just numbers  
- More advanced than `/stats`  
- Helps in long-term improvement  

---

## 📚 Summary
`/insights` = Smart report of your usage
