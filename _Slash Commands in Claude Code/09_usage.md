# ⚡ /usage — Claude Code Command

## 📌 Purpose
`/usage` is used to **check your usage, limits, and plan details** in Claude Code.

---

## ⚙️ What It Does
- Shows how much usage you have used  
- Displays your remaining limits  
- Shows token usage (input/output)  
- Displays your current plan (free or paid)  
- Helps track cost when using different models  

---

## 🚀 Usage
/usage

---

## 💡 What You Will See
- Total usage (how much you used)  
- Remaining quota (how much is left)  
- Model usage (Opus, Sonnet, Haiku)  
- Cost or token consumption  
- Plan limits (daily / monthly)  

---

## 💡 When to Use
- When you want to check remaining limits  
- When using expensive models (like Opus)  
- When working on long sessions  
- When managing cost and usage  

---

## 🧠 Pro Tip
Check `/usage` before starting a big task to avoid hitting limits in the middle of work.

---

## ⚠️ Notes
- Usage depends on your plan  
- More powerful models use more tokens  
- Long sessions increase usage  
- Helps avoid unexpected limits  

---

## 📚 Summary
`/usage` = Check usage, limits, and cost
