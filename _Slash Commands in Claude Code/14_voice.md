# ⚡ /voice — Claude Code Command

## ⚠️ Important Note
- `/voice` feature is not available for all users (limited rollout)  
- If `/voice` does not work, it means the feature is not enabled in your account  
- You can use **Windows Voice Typing (Win + H)** as a reliable alternative  
- Windows voice typing works faster and is more stable  

---

## 📌 Purpose
`/voice` is used to enable voice interaction so you can talk to Claude instead of typing.

---

## ⚙️ What It Does
- Turns voice mode ON  
- Lets you speak instead of typing  
- Converts speech into text  
- Allows hands-free interaction  

---

## 🚀 Usage
/voice

---

## 🎤 How to Use (Step-by-Step)
1. Type `/voice` and press Enter  
2. Press and hold the **Space bar**  
3. Start speaking  
4. Release Space bar to send your message  
5. Claude will respond normally  

---

## 💡 Alternative (Best Working Method)

### Windows Voice Typing
Win + H

👉 Press and speak  
👉 Your voice will convert into text instantly  

---

## 💡 When to Use
- When you don’t want to type  
- When explaining long ideas  
- When multitasking  
- When working faster  

---

## 🧠 Pro Tip
If `/voice` is not working, always use **Win + H** for better and stable results.

---

## ⚠️ Notes
- Requires microphone permission  
- Accuracy depends on voice clarity  
- May not work if feature is not enabled  

---

## 📚 Summary
`/voice` = Talk instead of typing
