# ⚡ @ (At Symbol)(at the rate) — Claude Code Usage

## 📌 Purpose
`@` is used to **reference files, folders, or context** inside Claude Code.

---

## ⚙️ What It Does
- Lets you point to a specific file  
- Helps Claude understand which file you are referring to  
- Adds file context into your prompt  
- Improves accuracy of responses  

---

## 🚀 Usage

### 📄 Reference a file
@filename.js

Example:
Explain this code @app.js

---

### 📂 Reference a folder
@src/

Example:
Review all files in @src/

---

### 🧠 Use in prompt
Fix bugs in @index.py  
Explain logic in @utils.js  

---

## 💡 When to Use
- When working with specific files  
- When you want accurate code explanations  
- When debugging code  
- When reviewing project files  

---

## 🧠 Pro Tip
Use `@file` instead of copying code manually — it saves time and keeps your prompt clean.

---

## ⚠️ Notes
- File must exist in your project  
- Works within Claude Code environment  
- Improves context understanding  

---

## 📚 Summary
`@` = Reference file or folder  

---

## 🇵🇰 اردو خلاصہ
`@` کا استعمال Claude Code میں کسی خاص فائل یا فولڈر کو واضح طور پر بتانے کے لیے کیا جاتا ہے تاکہ Claude کو صحیح context مل سکے۔  
جب آپ `@filename` لکھتے ہیں تو Claude اسی فائل کو دیکھ کر بہتر اور درست جواب دیتا ہے، جس سے explanation، debugging اور code review زیادہ آسان ہو جاتا ہے۔  
یہ طریقہ اس لیے ضروری ہے کیونکہ اس سے آپ کو code بار بار copy کرنے کی ضرورت نہیں پڑتی اور کام زیادہ منظم اور تیز ہو جاتا ہے۔
